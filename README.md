# Mini Social

A simple mini social network built with PHP.

## About the Project
This is a small prototype of a social network, created to demonstrate basic user functionality.  
Currently, it includes user registration, login, and a simple post system with basic interaction.

## Features
- User registration  
- User login  
- Pagination for posts  
- Edit page for posts  
- Post wall (text-only)  
- Default avatar for users  
- Main page displaying usersâ€™ posts  
- (No CSS or additional styling yet)  

## Technologies
- PHP  
- MySQL (via phpMyAdmin)  
- HTML  
- Runs on a local XAMPP server  

## Goal
To showcase a basic social network prototype and learn how to implement user registration, login, post management, and pagination in PHP.  
This project is intended for educational purposes.
